## Authentication

All API [endpoints](https://windermereservicescompany.mintlify.app/api-reference/agents) are authenticated using an API key and x-api-key header

```javascript
curl --request GET \
  --url https://api.windermere.com/v1/api
  --header 'x-api-key: <api-key>'
```
